Circle Radius


The Circle Radius function allows the operator to create a circle through two points by Selecting an existing point or Indicating a new point for either the center point, outer point or both. Additionally, the coordinates and radius of the circle may be Keyed via the Input Dialog (pictured below).

Step by Step


Step 1 of 2: Select curve or center point of the circle

The first step is define the center point of the circle. This may be accomplished in one of four ways:
  1. Select an existing circle
  2. Select an existing point
  3. Indicate a new point (free hand)
  4. Key the coordinates

Step 2 of 2: Drag to shrink/enlarge or Select the outer point of the circle

The final step is to define the radius of the circle. This may be accomplished in one of four ways:
  1. Select an existing offset point
  2. Select a new offset point (free hand)
  3. Drag the circle to the appropriate size (free hand)
  4. Key the radius

Optional Steps


The Input Dialog (pictured right) may be used to modify any of attributes of the circle prior to creating it.

These attributes include the layer, color, line pattern, or line thickness.
