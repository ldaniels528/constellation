Line Functions

Creation Functions

  • ANGLE - facilitates the creation of an angled line
  • HORIZONTAL - facilitates the creation of a horizontal line
  • NORMAL - facilitates the creation of a line that is normal (perpendicular) to a second line
  • PT-PT - facilitates the creation of a line between two points
  • RECTANGLE - facilitates the creation of a line between two points
  • TANGENT - facilitates the creation of a line that is tangent to a curve
  • VERTICAL - facilitates the creation of a vertical line

Modification Functions

  • FILLET - facilitates the joining of two lines
  • SPLIT - facilitates the breaking of a line into two new lines
  • TRIM - facilitates the trimming or extending of a line to a second line or point
