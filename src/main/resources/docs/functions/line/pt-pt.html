Line Point-to-Point


The Line Point-to-Point function allows the operator to create either a line segment or an infinite line between two points. The operator may either Select an existing point or Indicate a new point for either the starting point, ending point or both.

Step by Step


Step 1 of 2: Select or Indicate the start point of the line

The first step is to set the starting point of the line. This may be done in one of two ways:
  1. Selecting an existing point
  2. Indicating a new point (free hand)

Step 2 of 2: Select or Indicate the end point of the line

The final step is to set the ending point of the line. This may be done in one of two ways:
  1. Selecting an existing point
  2. Indicating a new point (free hand)

Optional Steps


The Input Dialog (pictured right) may be used to modify any of attributes of the line prior to creating it. These attributes include the layer, color, or line pattern.

Lines can be also be created by specifying their end-point coordinates via the dialog.
