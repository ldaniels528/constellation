Point Coordinates


The Point Coordinates function allows the operator to create a new point; either by Keying the coordinates via the Input Dialog (pictured below) or Indicating a new point by hand.

Step by Step


Step 1 of 1: Key the coordinates or Indicate the create a point

The first step is to set the starting point of the line. This may be done in one of two ways:
  1. Key the coordinates of the new point via the Input Dialog
  2. Indicate a new point (free hand)

Optional Steps


The Input Dialog (pictured right) may be used to modify any of attributes of the point prior to creating it. These attributes include the layer and color.

NOTE: The line pattern and line thickness attributes have no affect on points.
