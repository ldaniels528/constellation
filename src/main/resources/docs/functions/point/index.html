Points Functions

Coordinates/Projection

  • COORDS - facilitates the creation of a point at a user specified coordinate in space
  • OFFSET - facilitates the creation of a point whose location is an offset relative to an existing point.
  • PROJECT - facilitates the creation of a point whose location is the projection of an existing point onto a geometric entity

Intersection/Boundary

  • INTERSECT - facilitates the creation of an intersection point between two geometric entities
  • LIMITS - facilitates the creation of points along the boundary of a geometric entity
  • MID-POINT - facilitates the creation of a geometric entity-specific mid-point (e.g. center-point of a circle)
