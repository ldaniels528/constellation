About Constellation


Cost Effective

Constellation™ costs a fraction of what its competitors do, while providing all of the functionality that one would expect from a technical drawing application.


Easy to Use

Constellation™ is a two-dimensional drawing package that combines advanced geometry creation with the ease of use of a traditional drawing/paint application.


Extensible

Constellation™ offers a rich suite of tools that empower developers to extend the functionality of the system by allowing the creation of third-party functions and plug-ins.


Multi-platform

Constellation™ is a 100% native Java™ Application. As such it runs quite well on a number of platforms that support Java™ 1.5+ , and all three major platforms are officially supported.


  • Mac OS X® 10.5.x and later
  • Windows® XP™ and Vista™
  • GNU Linux 2.6.x (Debian, RedHat®, Ubuntu®)
