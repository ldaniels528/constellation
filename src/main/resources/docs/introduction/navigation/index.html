Basic Navigation


Selection

Constellation™ offers two methods of selection:
  1. Selection of an existing element via the left mouse button .
  2. Indication of a new element (usually a point) via the right mouse button .

Panning

Panning is invoked by dragging via the left mouse button . Dragging left results in the camera panning left. Dragging right results in the camera panning right.

Zooming

Zooming is invoked by scrolling via the mouse wheel ; forward scrolling results in zooming in, backward scrolling result in zooming out.
